
# Build

`./gradlew build docker`

# Run
Run `docker run -p 8080:8080 -t com.example/demo-docker`
See also [docker plugin](https://spring.io/guides/gs/spring-boot-docker/#_summary)

# Urls
http://localhost:8080/
http://localhost:8080/jdbc
